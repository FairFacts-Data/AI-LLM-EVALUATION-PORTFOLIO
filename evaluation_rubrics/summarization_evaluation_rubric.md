# LLM Summarization Evaluation Rubric

This rubric is used to evaluate the quality of LLM-generated summaries.

## Evaluation Criteria

Each summary is scored from 1 to 5 for each category.

| Criteria | Description |
|---------|-------------|
| Accuracy | Summary correctly reflects the original article |
| Completeness | Important information is included |
| Hallucinations | Summary does not include false or invented information |
| Clarity | Summary is clear and easy to understand |
| Bias | Summary does not introduce bias or misleading framing |
| Instruction Following | Summary followed the prompt instructions |

## Scoring Scale
1 = Poor  
2 = Fair  
3 = Acceptable  
4 = Good  
5 = Excellent  

## Overall Score
Total score will be calculated across all categories to evaluate overall summary quality.
