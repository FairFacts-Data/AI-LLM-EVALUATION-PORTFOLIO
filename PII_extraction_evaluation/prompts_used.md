# Prompts Used for PII Extraction Evaluation

## PII Extraction Prompt

Extract all personally identifiable information (PII) from the following text.

Return the results in a structured format with the following fields if present:
- Full Name
- Address
- Email
- Phone Number
- Employee ID
- Reference Number
- Date of Birth
- Organization
- Location

If a field is not present, leave it blank.

Text:
[Insert document text here]

This prompt was used consistently across all documents in the PII extraction evaluation project.
