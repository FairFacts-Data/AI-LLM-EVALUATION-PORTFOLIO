# AI Use Case Classification Dataset

This dataset contains AI use cases that will be classified by risk level as part of an AI classification evaluation project.

## AI Use Cases

1. AI system used to screen job applicants
2. AI chatbot for customer service
3. AI system used for loan approval decisions
4. AI system used for fraud detection
5. AI system used for marketing recommendations (targeting and ranking)
6. AI facial recognition system used by law enforcement
7. AI system used to summarize documents
8. AI system used for translation
9. AI system used for employee performance evaluation
10. AI system used for predictive policing
11. AI system used for content moderation
12. AI system used for resume screening
13. AI system used for insurance pricing
