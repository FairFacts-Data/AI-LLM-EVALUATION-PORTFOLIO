# Classification Evaluation Project

This project evaluates how well a large language model (LLM) can classify AI use cases into risk categories such as High Risk, Medium Risk, Low Risk, and Minimal Risk.

## Objective
Evaluate the accuracy and reasoning of AI risk classification for different AI use cases.

## Method
AI use cases are classified using a consistent prompt. The classifications are evaluated using a structured evaluation rubric to assess accuracy, reasoning, and consistency.

## Skills Demonstrated
- Prompt engineering
- LLM classification evaluation
- AI risk classification
- Human evaluation of AI outputs
- Evaluation rubric development
- Responsible AI and risk assessment concepts
