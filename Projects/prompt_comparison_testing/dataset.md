# Dataset

## Source Document

This evaluation used the following article as the dataset for prompt comparison testing:

Meta Engineering Blog – Adaptive Ranking Model
https://engineering.fb.com/2026/03/31/ml-applications/meta-adaptive-ranking-model-bending-the-inference-scaling-curve-to-serve-llm-scale-models-for-ads/

## Dataset Description

The article describes Meta’s Adaptive Ranking Model, which is designed to enable LLM-scale recommendation models to run efficiently in real-time advertising systems. The article includes technical architecture details, business impact, infrastructure design, and system-level innovations.

This document was selected because it contains both technical and business content, making it suitable for testing summarization, information extraction, explanation, and analysis prompts.

## Purpose of Dataset

The purpose of using this dataset was to evaluate how different prompt instructions affect the quality, structure, and usefulness of large language model outputs when applied to the same source material.
