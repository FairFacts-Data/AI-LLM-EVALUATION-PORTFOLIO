# Prompt Comparison Testing

## Objective

The objective of this project is to evaluate how different prompt instructions affect the quality, accuracy, and usefulness of large language model (LLM) outputs for the same task. This evaluation focuses on prompt design and its impact on model performance.

## Task

The task used in this evaluation is document summarization and information extraction using multiple prompt variations applied to the same input text.

## Dataset

This project uses sample documents and text passages to evaluate how different prompts influence the structure, completeness, and accuracy of model outputs.

## Prompts Tested

Multiple prompt variations were tested, including:

* Basic summarization prompt
* Bullet point summary prompt
* Executive summary prompt
* Risk-focused summary prompt
* Action-oriented summary prompt
* Information extraction prompt

## Evaluation Method

Outputs were evaluated using a structured scoring rubric and manual review. Each output was evaluated for accuracy, completeness, clarity, hallucination, format adherence, and usefulness.

## Scoring Rubric

Each output was scored on a scale (e.g., 1–5) across the following criteria:

* Accuracy
* Completeness
* Clarity
* Hallucination
* Format Adherence
* Usefulness
