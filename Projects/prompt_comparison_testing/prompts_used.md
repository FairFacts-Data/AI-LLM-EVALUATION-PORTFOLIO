# Prompts Used

The following prompts were used for prompt comparison testing:

1. Summarize the article.
2. Summarize the article in 5 bullet points.
3. Write an executive summary of the article.
4. Extract the key technical innovations described in the article.
5. Explain the article in simple terms for a non-technical audience.
6. Identify the main business benefits described in the article.
7. Identify the technical challenges described in the article.
8. Provide key takeaways from the article.

These prompts were selected to test how different prompt instructions affect output structure, detail level, audience targeting, and usefulness.
