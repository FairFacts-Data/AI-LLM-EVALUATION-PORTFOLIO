# Summarization Evaluation Project

This project evaluates large language model (LLM) summarization performance.

## Objective
Evaluate the quality of AI-generated summaries for accuracy, completeness, clarity, and hallucinations.

## Method
Articles are summarized using an LLM and evaluated using a structured evaluation rubric.

## Evaluation Criteria
- Accuracy
- Completeness
- Hallucinations
- Clarity
- Bias
- Instruction Following

## Files in This Folder
- Prompts used
- Articles used
- Generated summaries
- Evaluation scores
- Results summary
