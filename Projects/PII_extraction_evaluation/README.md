# PII Extraction Evaluation Project

This project evaluates how well a large language model (LLM) can identify and extract personally identifiable information (PII) from text documents.

## Objective
Evaluate the accuracy of AI extraction of PII such as names, addresses, emails, phone numbers, employee IDs, and dates of birth.

## Method
Sample documents containing PII were created and the LLM was prompted to extract structured PII fields. The extracted information was evaluated for accuracy, completeness, and formatting.

## Skills Demonstrated
- PII detection and extraction
- Structured output evaluation
- AI privacy and data governance use cases
- Prompt engineering
- Human-in-the-loop AI evaluation
- Evaluation rubric development
